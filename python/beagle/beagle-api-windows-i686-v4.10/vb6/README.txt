                   Total Phase Beagle Sample Code
                   ------------------------------

Introduction
------------
This directory contains examples that use the Beagle Visual Basic
language bindings.


Contents
--------
See top level EXAMPLES.txt for descriptions of each example.


Instructions
------------
1) Copy 'beagle.dll' to the WINDOWS/System32 folder

2) Open the "examples" project in Visual Basic 6.0

3) Make sure that the Immediate window is displayed

4) Run the project

5) Use the supplied GUI form to execute the examples
